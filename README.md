# Tour by Bihar

Tour by Bihar is a tourism information web application that provides district-wise details of tourist places in a structured and easy-to-use format. Users can explore historical sites, religious places, parks, lakes, and cultural attractions through a simple interface designed to promote tourism and help travelers discover new destinations.

## Features

* District-wise tourist place listing
* Historical, religious, cultural, and natural attractions
* Simple and user-friendly interface
* Interactive map-based navigation

## Technologies Used

* Vite
* TypeScript
* React
* Tailwind CSS
* shadcn-ui

## How to Run Locally

```sh
npm install
npm run dev
```

## Project Goal

The goal of this project is to provide organized tourism information so that users can easily discover and explore different tourist destinations.
